# 🌌 NASA Mission Explorer

## 📌 Overview

NASA Mission Explorer is a web application that allows users to explore space through NASA’s Astronomy Picture of the Day (APOD). Users can select any date to view high-quality images and learn interesting facts about the universe.

---

## 🎯 Objective

The goal of this project is to demonstrate:

* JavaScript fundamentals
* API integration using `fetch`
* Dynamic rendering of data on the UI
* Use of Array Higher-Order Functions
* Responsive and user-friendly design

---

## 🔗 Public API Used

* **NASA APOD API**
  https://api.nasa.gov/

---

## ✨ Planned Features

### 🔍 Core Features

* 📅 Date-based search to view images from any day
* 🖼️ Display image, title, and description dynamically
* ⏳ Loading indicator while fetching data
* ❌ Error handling for invalid dates or API issues

### ⚙️ Advanced Features (Milestone 3)

* 🔎 Search functionality
* 🎯 Filter saved/favorite images
* 🔃 Sort images by date
* ❤️ Add/remove favorites (using localStorage)
* 🌙 Dark/Light mode toggle
